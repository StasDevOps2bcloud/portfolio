Create_Shortcuts_To_Start_Stop_Azure_VM.ps1
This script was developed to solve a problem with always running Azure VMs, which were only used intermittently by Finance stuff. These VMs were accruing unnecessary costs 24/7. My solution was to create an Azure Automation Runbook, which would Start/Stop Azure VM. This script was used to create shortcut on windows devices, which when clicked would trigger the automation runbook via a webhook with an authorization token.
